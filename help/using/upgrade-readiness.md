# Controlelijst voor gereedheid voor upgrade

Gebruik de volgende keuzelijst om u te helpen bij het begrijpen en voorbereiden van een upgrade van [!DNL Dynamic Media Classic] naar [!DNL Dynamic Media] .

|  | Taak | Beschrijving |
| :--- | :--- | --- |
| **Fase 1: Verlenen van vergunningen** | Uitvoeren van contract | Op basis van verkeer en opslag werkt het Adobe-accountteam met u samen om over te schakelen van de [!DNL Dynamic Media Classic] -licentie om de [!DNL Dynamic Media] -licentie te vernieuwen. |
| **Fase 2: Voorbereiding** | Functiegebruik valideren | Controleer of de functies die worden gebruikt in [!DNL Dynamic Media Classic] beschikbaar zijn in [!DNL Dynamic Media] . Zie de [ pagina van de Vergelijking van de Eigenschap ](/help/using/upgrade-feature-comparison.md). De belangrijkste mogelijkheden die nog niet beschikbaar zijn in [!DNL Dynamic Media] omvatten het volgende: <br>・ Visuele Configurator (de Auteur van het Beeld, de Render van het Beeld).<br>・ Afbeeldingssjablonen (1 :1 Sjablonen).<br>・ eCatalogs.<br> als de bovengenoemde eigenschappen worden gebruikt, kan de verbetering nog met de veronderstelling voorkomen dat die eigenschappen door middel van [!DNL Dynamic Media Classic] toegankelijk zouden zijn. |
|   | Elementen identificeren | Elementen en voorinstellingen zoeken en klaar maken om te worden gebruikt voor een upgrade. |
| **Fase 3: Milieu** | Upgrade uitvoeren [!DNL Adobe Experience Manager] | Alle instanties van [!DNL Adobe Experience Manager] moeten worden bijgewerkt naar de meest recente versie. |
|   | Instellen [!DNL Dynamic Media] | Adobe Consulting of Partner configureert [!DNL Dynamic Media] met uw referenties. |
| **Fase 4: Verbetering** | Elementen repliceren | Tijdens het upgradeproces worden de aangewezen [!DNL Dynamic Media Classic] -elementen overgenomen in Dynamic Media. |
| **Fase 5: De administratieve Opstelling** | Gebruikers en machtigingen instellen | Maak gebruikers en geef de juiste machtigingen. |
|   | Videocoderingsprofielen instellen | Maak videocoderingsprofielen. |
|   | Voorinstellingen viewer instellen | Viewer-voorinstellingen maken. |
|   | Voorinstellingen afbeelding instellen | Voorinstellingen voor afbeeldingen instellen. |
| **Fase 6: Bevestiging** | Validatie | Verifieer gebruiksgevallen, activa, verbindingen, en APIs. |
